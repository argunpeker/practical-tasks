# Booking.com Testing Analysis

This repository contains my solution for a practical testing task assigned during my internship process at EPAM.

The goal of this task is to:

* Break down a complex real-world application (Booking.com) into its core features and functionalities
* Identify appropriate testing types, methods, and levels for each feature

The project includes:

* A structured feature-based analysis
* A visual mind map representation
* A checklist-driven testing approach

This work reflects my understanding of software testing fundamentals, including functional decomposition, test strategy design, and structured thinking.
